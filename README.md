# Калькулятор кредита та депозиту - Single Page Application

## Опис проекту

Це веб-застосунок, розроблений як односторінковий додаток (SPA - Single Page Application), який дозволяє розраховувати параметри кредитів та депозитів. Застосунок надає користувачам можливість вводити фінансові дані, отримувати детальні розрахунки та зберігати результати у текстовий файл.

**Lab 6 - Лабораторна робота з розробки Single Page Application**

---

## Основні функціональні можливості

### 1. **Два режими розрахунку**
   - **Режим Кредиту**: розрахунок місячних платежів та загальної суми виплат
   - **Режим Депозиту**: розрахунок нарахованих відсотків та прибутку

### 2. **Вхідні параметри**
   - **Сума**: розмір кредиту або депозиту (грн)
   - **Річна процентна ставка**: відсоток (%)
   - **Період**: тривалість (місяці)
   - **Період капіталізації**: щомісячна, щоквартальна, щорічна

### 3. **Результати розрахунків**
   - Початкова сума
   - Нараховані відсотки
   - Загальна сума
   - Місячний платіж
   - Деталізований розрахунок у вигляді таблиці

### 4. **Дії з результатами**
   - 💾 **Зберегти результати**: експорт у текстовий файл з форматуванням
   - 🖨️ **Друкувати**: роздрукування звіту
   - 🔄 **Новий розрахунок**: повернення до форми для нових обчислень

## Алгоритми розрахунку

### **Розрахунок Кредиту**
Використовується формула аннуїтету для рівномірних місячних платежів:

```
M = P × [r(1+r)^n] / [(1+r)^n - 1]
```

Де:
- M = місячний платіж
- P = сума кредиту
- r = місячна процентна ставка
- n = кількість місяців

### **Розрахунок Депозиту**
Використовується формула складних відсотків:

```
A = P × (1 + r/n)^(nt)
```

Де:
- A = виділена сума
- P = початкова сума
- r = річна процентна ставка
- n = кількість капіталізацій на рік
- t = кількість років

## Запуск застосунку

1. **Відкрийте файл `index.html` в браузері**
   - Клацніть подвійно на файлі, або
   - Перетягніть його в браузер

2. **Або використайте локальний сервер** (рекомендується)
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Node.js
   npx http-server
   ```
   Потім відкрийте `http://localhost:8000`
